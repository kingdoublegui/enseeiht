Package org.eclipse.emf.cdo.transfer.ui.swt

Reusable controls for dealing with transfers of resources between systems.

  • Class Summary 
    Class Description
    TransferComposite
    A vertical sash that contains a TransferTreeComposite and a TransferDetailsComposite.
    TransferDetailsComposite
    A composite that lays out controls for the details of a single CDOTransferMapping.
    TransferDetailsComposite.UnmappedModelsContentProvider
    A reusable content provider for the unmapped models of a transfer.
    TransferDetailsComposite.UnmappedModelsLabelProvider
    A reusable label provider for the unmapped models of a transfer.
    TransferTreeComposite
    A composite that contains a tree viewer for the mappings of a transfer.
    TransferTreeComposite.UnmapAction
    An action to unmap a transfer mapping.
